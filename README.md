# 나 사용법

**나를 이해하는 8가지 이야기**

8가지 질문에 답하고, 나만의 사용법을 만들어 공유해보세요.

[**→ 지금 시작하기**](https://how-to-use-i.vercel.app)

</div>

## ✨ 주요 기능

- **카카오 로그인** — 별도 회원가입 없이 카카오로 시작
- **8가지 질문** — 나를 소개하는 질문에 차근차근 답변
- **공유 페이지** — 나만의 URL로 누구에게나 공유
- **공유 설명서** — 함께 읽으며 서로를 더 깊이 알아가는 방법 안내

## 💬 8가지 질문

> 나를 가장 잘 설명하는 질문들을 골랐어요.

1. 나에게 성공이란?
2. 나를 동기부여 시키는 것은?
3. 나의 커뮤니케이션 스타일은?
4. 상대를 불편하게 만들 수도 있는 나만의 성격은?
5. 나에게 신뢰를 얻거나 잃게 만드는 성격은?
6. 나만의 강점은?
7. 발전시키고 싶은 역량이 하나 있다면?
8. 살면서 느낀 가장 큰 성취감은?

## 🔄 서비스 흐름

```
카카오 로그인  →  8가지 질문 답변  →  나만의 URL 완성  →  공유
```
