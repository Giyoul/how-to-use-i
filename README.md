# 나 사용법

**나를 이해하는 8가지 이야기**

8가지 질문에 답하고, 나만의 사용법을 만들어 공유해보세요.

[**→ 지금 시작하기**](https://how-to-use-i.vercel.app)

## ✨ 주요 기능

- **카카오 로그인** — 별도 회원가입 없이 카카오로 시작
- **8가지 질문** — 나를 소개하는 질문에 차근차근 답변
- **공유 페이지** — 나만의 URL로 누구에게나 공유
- **공유 설명서** — 함께 읽으며 서로를 더 깊이 알아가는 방법 안내

## 🛠 기술 스택

| | |
|---|---|
| Framework | Next.js 16 (App Router) |
| Styling | Tailwind CSS v4 |
| Auth & DB | Supabase |
| Deploy | Vercel |
